Detailed Description

The OntologyClient class provides an abstraction common to all ontologenius exploration ROS services.

The OntologyClient implement the functions common to every ontologenius exploration.

This class is based on ClientBase and so ensure a persistent connection with the service based on. The persistent connection ensure a minimal response time. A reconnection logic is implemented in the event that the persistent connection fails.

See also IndividualClient, ClassClient, DataPropertyClient and ObjectPropertyClient.

std::vector<std::string> OntologyClient::getUp(const std::string& name, int depth = -1)

Gives all concepts below the one given in the parameter: name.

The optional depth parameter can be set to limit tree propagation to a specific value. The default value -1 represents no propagation limitation.
